Síntesis

La ontología de Cubo de Datos Turísticos describe la estructura que contiene las dimensiones y medidas que pueden ser utilizadas para generar cubos de datos turísticos. Algunos ejemplos de cubos son los de ocupación turística, flujo turístico y los de indicadores que permiten evaluar aspectos como ocupación promedio, crecimiento en número de plazas, crecimiento de flujo turístico, etc. Para desarrollar el cubo de datos turísticos se reutiliza el vocabulario W3C Data Cube.
