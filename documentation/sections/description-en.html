Cubo de Datos Turísticos: Description back to ToC

Figure 1 shows the conceptual model of the tourism cube with its classes, relationships (object properties), and some individuals.

Cubes are defined using the concepts and properties described in the W3C RDF Data Cube vocabulary. The DSD_TourismOccupation structure defines the set of cube dimensions and properties that can be combined in different ways to generate cubes that serve various purposes.

For example, a cube that analyzes tourism occupancy data could use the administrative region dimension and the period dimension. The measure would be the number of travelers which refers to the number of people who have occupied accommodation establishments in a certain period of time, the year 2025, in a certain administrative region, the municipality of Benidorm. Similarly, by adding the dimensions type of tourist flow and country of residence, it is possible to have the (number of tourists) from a certain country, for example Germany, who have arrived in Benidorm in the year 2025.
