Introduction back to ToC

This ontology defines a set of dimensions and measures for tourism cubes to facilitate the statistical analysis of tourism data. Several cubes are presented that combine dimensions and measures related to tourist occupancy, tourist flow, and indicators that allow for the evaluation of aspects such as accommodation capacity utilization, tourist flow intensity, and the growth in the number of available beds. The cubes are defined using the concepts and properties described in the W3C Data Cube vocabulary. To represent the required dimensions and measures, several vocabularies have been reused, and SKOS code lists have been defined. This ontology is part of the catalog of ontology being developed within the context of the Data Space for Smart Urban Infrastructures (EDINT). The representation of primary tourism data is done through the ontology of SEGITTUR as described in the EDINT Tourism Ontology.
